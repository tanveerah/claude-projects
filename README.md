## Tic Tac Toe

A clean, fully playable Tic Tac Toe game that runs in any web browser — no installation or dependencies required.

### Features

- Two-player mode (X and O, same device)
- Win detection with animated cell highlights
- Draw detection
- Persistent scoreboard across rounds
- New Game button to reset the board without clearing scores

### How to Play

1. Open `tictactoe.html` in any web browser
2. Players take turns clicking empty cells
3. First to get three in a row (horizontal, vertical, or diagonal) wins
4. Click **New Game** to start a new round — scores carry over

### How It Was Created

This game was built entirely using **[Claude Code](https://claude.com/code)**, Anthropic's AI coding assistant, in a single prompt. Claude generated the complete HTML, CSS, and JavaScript as one self-contained file with no external dependencies.

The project was then version-controlled and published to GitHub — also via Claude Code — using the GitHub CLI (`gh`).

**Stack:** HTML5 · CSS3 · Vanilla JavaScript
